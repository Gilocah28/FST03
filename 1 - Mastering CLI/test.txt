Ls 

*See Current Directory*

pwd

*Go To Directory*

cd

*Return Directory*

cd ..

*Create file or new file*

touch test.txt

*Remove file*

rm test.txt

*Login GitHub account*

git config --global [user.name](http://user.name/) "username"

*See your git user name*

git config [user.name](http://user.name/)

git config --global user.email "email”

*See the Email*

git config [user.](http://user.name/)email

*initialize the repo*

git init

*Checked the Status*

git status
